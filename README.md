# TRI-TUE-NHAN-TAO---UEH
tổng hợp các bài toán thực hành về Phân tích Dữ liệu Không gian và ứng dụng Machine Learning trong việc giải quyết các bài toán thực tế về logistics đô thị, quy hoạch giao thông và tối ưu hóa chuỗi cung ứng.  Dự án sử dụng Python để xử lý, trực quan hóa và xây dựng các mô hình AI trực tiếp trên nền tảng bản đồ số (đặc biệt là khu vực TP.HCM).
